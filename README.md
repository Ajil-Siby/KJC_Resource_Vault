# KJC AI Resource Vault
**A Full-Stack Knowledge Management Tool for AI/ML Students**

## Overview
This project is a single-page web application designed for students at **Kristu Jayanti College** to organize high-quality AI and Machine Learning resources. It fulfills the requirements for the SEC Web Development Project by demonstrating a modern full-stack workflow.

## Features
* **Full-Stack Architecture:** Built using Python (Flask) for the backend and MongoDB Atlas for the cloud database.
* **Modern Workflow:** Implemented using version control (Git) and hosted on the cloud (Render).
* **AI-Ready:** Designed specifically for the AI & ML Department to store technical study materials.

## Technical Stack
* **Frontend:** HTML5, CSS3 (Bootstrap 5)
* **Backend:** Python (Flask)
* **Database:** MongoDB Atlas (Cloud)
* **Hosting:** Render
* **Version Control:** Git & GitHub
